Reflections is a command-line tool for Solidity analysis built in Rust.

This project provides a framework for analyzing Solidity smart contracts with various commands and tools.

## Installation (standalone)

Reflections can be installed with:

```bash
cargo install reflections
```

### Verify installation

```bash
reflections help
```

## Compile from Source

Clone this repository, then run `cargo build --release` inside the root.

The `reflections` binary will be located inside the `target/release/` folder.
